# Реализация расширения Telegram


## Функциональность


1. **Парсинг чатов и генерация резюме**
   - Извлекает сообщения из активного чата Telegram
   - Отправляет сообщения в OpenAI API для генерации резюме
   - Поддерживает несколько селекторов сообщений для совместимости

2. **Динамическое отображение резюме**
   - Заменяет статический заполнитель сгенерированными резюме
   - Показывает состояния загрузки во время вызовов API
   - Отображает сообщения об ошибках при сбое API

3. **Автообновление при смене чата**
   - Отслеживает изменения URL и мутации DOM
   - Автоматически генерирует новые резюме при переключении чатов
   - Очищает предыдущие резюме при смене чатов

4. **Состояния загрузки и обработка ошибок**
   - Анимированный спиннер во время запросов API
   - Подробные сообщения об ошибках
   - Предотвращение нескольких одновременных запросов

### ✅ Дополнительные функции (реализовано 2)

1. **История резюме**
   - Хранит до 50 предыдущих резюме локально
   - Показывает время, ID чата и количество сообщений
   - Очистка истории одним кликом

2. **Перетаскиваемое окно**
   - Полностью перетаскиваемый интерфейс расширения
   - Сохраняет позицию между сессиями
   - Ограничивает перемещение границами viewport
   - Функциональность сворачивания/разворачивания

## Ключевые детали реализации

### Стратегия парсинга чатов
- Использует множественные DOM-селекторы для поиска сообщений
- Извлекает автора, текст и временную метку из каждого сообщения
- Обрабатывает различные макеты интерфейса Telegram и обновления

### Интеграция с OpenAI
- Настраиваемое хранение API-ключа в localStorage
- Правильная обработка ошибок API
- Контекстные промпты для лучших резюме

### Управление состоянием
- React хуки для состояния компонентов
- localStorage для сохранения данных
- Правильная очистка наблюдателей

### Пользовательский опыт
- Плавные анимации и переходы
- Интуитивный интерфейс с вкладками
- Адаптивный дизайн внутри окна расширения

## Соображения безопасности

- API-ключ хранится в localStorage (не в коде)
- Правильная валидация ввода
- Обработка CORS для OpenAI API
- Отсутствие секретов в репозитории

## Инструкции по тестированию

1. Сборка расширения:
   ```bash
   yarn install
   yarn build
   ```

2. Загрузка в Chrome:
   - Перейдите в `chrome://extensions/`
   - Включите "Режим разработчика"
   - Нажмите "Загрузить распакованное расширение"
   - Выберите папку `build/`

3. Настройка:
   - Посетите https://web.telegram.org
   - Введите ваш API-ключ OpenAI в расширении
   - Выберите любой чат для генерации резюме

- Очистка слушателей событий

## Известные ограничения

1. **Изменения интерфейса Telegram**: Может потребоваться обновление при изменении DOM-структуры Telegram
2. **Лимиты API**: Нет встроенного ограничения скорости для OpenAI API
3. **Длина сообщений**: Очень длинные чаты могут достигнуть лимитов токенов OpenAI
4. **Поддержка языков**: Язык резюме зависит от языка чата
