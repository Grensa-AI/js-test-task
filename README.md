## Описание изменений

Было реализовано браузерное расширение для Telegram Web, которое:
- собирает 20 сообщений из верхней части открытого чата, включая отправителя, текст и время. Сообщения берутся в порядке появления в DOM — чаще всего это более старые сообщения, если пользователь прокручивал чат вверх;
- отправляет эти сообщения в API Cohere, чтобы сгенерировать сжатое резюме диалога;
- отображает полученное резюме в окне в правом верхнем углу экрана;
- отслеживает смену чатов и сбрасывает старое резюме при переключении;
- показывает спиннер во время генерации и сообщение об ошибке при сбое (например, если нет интернета или отсутствует API-ключ).



## Инструкции по тестированию

1. Соберите расширение: `yarn build`
2. Загрузите в Chrome из папки `build/`
3. Откройте https://web.telegram.org
4. Убедитесь, что появилось окно с заголовком "Grensa.AI"
5. Прокрутите активный чат до нужного места
6. Нажмите кнопку "Сгенерировать резюме"
7. Проверьте сброс резюме при смене чата
8. Проверьте отображение ошибок при пустом чате или отсутствии API-ключа

##  Настройка API-ключа Cohere

1. Перейдите на сайт [https://dashboard.cohere.com/api-keys](https://dashboard.cohere.com/api-keys)
2. Зарегистрируйтесь или войдите в аккаунт.
3. Сгенерируйте новый API-ключ или скопируйте уже созданный пробный ключ.
4. В корне проекта создайте файл `.env`
5. Добавьте в файл следующую строку:

```env
REACT_APP_COHERE_API_KEY=ваш_ключ
```
