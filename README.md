# Расширение Grensa.AI для Telegram Web (Тестовое задание)

Браузерное расширение, которое анализирует чаты Telegram и генерирует краткие резюме с помощью ИИ (OpenAI, OpenRouter, Gemini).

## Реализованные функции

### Основные задачи (обязательные)

- **Парсинг чата и генерация резюме через ИИ API**
  - Извлечение сообщений из активного чата Telegram Web
  - Поддержка множественных провайдеров: OpenAI, OpenRouter, Gemini
  - Умная обработка ошибок API с понятными сообщениями

- **Отображение резюме в окне расширения**
  - Красивое форматирование с поддержкой Markdown
  - Адаптивный дизайн с современным UI

- **Автоматическое обновление при смене чата**
  - Отслеживание изменений URL и DOM
  - Настраиваемое автообновление с задержкой
  - Предотвращение дублирования запросов

- **Состояние загрузки с индикатором**
  - Анимированный спиннер во время генерации
  - Блокировка повторных запросов
  - Детальная индикация ошибок

### Дополнительные задачи (реализованы)

- **История резюме**
  - Локальное хранение в localStorage
  - Интерфейс с табами для просмотра истории
  - Возможность удаления, копирования, экспорта в TXT
  - Автосохранение с настройками

- **Перетаскиваемое окно расширения**
  - Полная реализация drag & drop
  - Сохранение позиции между сессиями
  - Ограничение перемещения границами экрана

- **Кнопки скрытия/показа расширения**
  - Плавающая кнопка для открытия/скрытия
  - Сохранение состояния и контекста
  - Toggle функциональность

### Дополнительные улучшения

- **Поддержка множественных провайдеров ИИ** (OpenAI, OpenRouter, Gemini)
- **Безопасное хранение API ключей** в session storage
- **Современный UI/UX** с градиентами и анимациями
- **Детальная обработка ошибок** с понятными сообщениями
- **Экспорт истории** в текстовый файл
- **Автосохранение настроек** пользователя

## Как работает расширение

### Архитектура

Расширение состоит из трёх основных частей:

1. **Content Script** (`content.js`) - внедряется в страницу Telegram Web
   - Отслеживает изменения чатов
   - Парсит сообщения из DOM
   - Отправляет данные в background script

2. **Background Script** (`background.js`) - работает в фоне
   - Обрабатывает API запросы к ИИ провайдерам
   - Управляет настройками и API ключами
   - Координирует работу между компонентами

3. **Popup Window** (`App.js`) - пользовательский интерфейс
   - Отображает резюме и историю
   - Управляет настройками
   - Обрабатывает пользовательские действия

### Процесс работы

1. **Инициализация**: При открытии Telegram Web content script начинает отслеживать изменения
2. **Парсинг чата**: При смене чата извлекаются последние сообщения (до 50)
3. **Генерация резюме**: Данные отправляются в ИИ API через background script
4. **Отображение**: Результат форматируется и показывается в popup окне
5. **Сохранение**: Резюме автоматически сохраняется в историю

### Технические особенности

- **Перетаскиваемое окно**: Реализовано через обработчики событий мыши
- **Автообновление**: Отслеживание URL и DOM изменений с debounce
- **Безопасность**: API ключи хранятся только в session storage
- **Обработка ошибок**: Детальные сообщения для разных типов ошибок API

## Установка и настройка

### Предварительные требования

- Node.js версии 16 или выше
- yarn
- Браузер Chrome или любой Chromium-браузер
- Аккаунт OpenAI/OpenRouter/Gemini для получения API ключа

### Установка зависимостей

```bash
# Клонируйте ваш форк репозитория
git clone https://github.com/mrKamanov/js-test-task.git
cd js-test-task
yarn install
```

### Сборка расширения

```bash
yarn build
```

### Загрузка в браузер

1. Откройте `chrome://extensions/`
2. Включите "Режим разработчика"
3. Нажмите "Загрузить распакованное расширение"
4. Выберите папку `build/`

### Настройка API ключа

1. Откройте настройки расширения
2. Введите API ключ и выберите провайдера
3. Сохраните настройки

1. Откройте https://web.telegram.org

6. Проверьте историю во вкладке "История"
