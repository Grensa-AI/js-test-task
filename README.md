# Grensa.AI - Telegram Chat Analyzer

Браузерное расширение для Chrome, которое анализирует чаты в Telegram Web и генерирует краткие резюме с помощью OpenAI API.

## ✨ Возможности

### ✅ Реализованные функции

- **Парсинг чата и генерация резюме через OpenAI API** - Автоматический анализ сообщений в чате
- **Отображение резюме в окне расширения** - Красивое и информативное отображение результатов
- **Автоматическое обновление при смене чата** - Отслеживание изменений и генерация нового резюме
- **Состояние загрузки с индикатором** - Визуальная индикация процесса анализа
- **Безопасное хранение API ключа** - Локальное хранение в chrome.storage
- **Настройки API ключа** - Удобный интерфейс для настройки OpenAI API
- **Валидация API ключа** - Проверка корректности введенного ключа
- **Обработка ошибок** - Информативные сообщения об ошибках
- **Адаптивный дизайн** - Современный UI с использованием styled-components

## 🚀 Установка и настройка

### Предварительные требования

- Node.js версии 16 или выше
- npm или yarn
- Браузер Chrome или любой Chromium-браузер
- Аккаунт OpenAI для получения API ключа

### 1. Клонирование и установка

```bash
git clone https://github.com/YOUR_USERNAME/js-test-task
cd js-test-task
npm install
```

### 2. Сборка расширения

```bash
npm run build
```

Собранные файлы будут находиться в папке `build/`.

### 3. Загрузка расширения в браузер

1. Откройте Chrome/Chromium браузер
2. Перейдите по адресу `chrome://extensions/`
3. Включите "Режим разработчика" в правом верхнем углу
4. Нажмите "Загрузить распакованное расширение"
5. Выберите папку `build/` из вашего проекта
6. Расширение будет установлено и активировано

### 4. Настройка OpenAI API ключа

1. Перейдите на [platform.openai.com](https://platform.openai.com/)
2. Создайте аккаунт или войдите в существующий
3. Перейдите в раздел "API Keys"
4. Нажмите "Create new secret key"
5. Скопируйте ключ
6. Откройте расширение и перейдите в настройки (⚙️)
7. Вставьте API ключ и нажмите "Сохранить"

## 📖 Использование

### Основное использование

1. Откройте [web.telegram.org](https://web.telegram.org/)
2. Войдите в свой аккаунт Telegram
3. Откройте любой чат с сообщениями
4. Нажмите на иконку расширения в панели браузера
5. Расширение появится в правом верхнем углу экрана
6. Дождитесь завершения анализа чата
7. Прочитайте сгенерированное резюме

### Функции интерфейса

- **Кнопка "Обновить"** - Перегенерировать резюме для текущего чата
- **Кнопка настроек (⚙️)** - Открыть настройки API ключа
- **Автоматическое обновление** - При смене чата или появлении новых сообщений

## 🛠️ Техническая архитектура

### Структура проекта

```
├── public/
│   ├── manifest.json       # Конфигурация расширения
│   ├── index.html         # HTML шаблон
│   └── icon.svg           # Иконка расширения
├── src/
│   ├── App.js             # Главный компонент
│   ├── content.js         # Скрипт для внедрения в страницу
│   ├── background.js      # Фоновый скрипт
│   ├── services/
│   │   ├── openaiService.js    # Сервис для работы с OpenAI API
│   │   └── chatParser.js       # Сервис для парсинга чатов
│   └── Components/
│       ├── Title/         # Компонент заголовка
│       ├── Summary/       # Компонент резюме
│       └── Settings/      # Компонент настроек
├── package.json
└── craco.config.js        # Конфигурация сборки
```

### Ключевые компоненты

#### openaiService.js
- Безопасное хранение API ключа в chrome.storage
- Генерация резюме через OpenAI API
- Валидация API ключа
- Обработка ошибок API

#### chatParser.js
- Парсинг сообщений из DOM Telegram Web
- Отслеживание изменений в чате
- Определение смены чата
- Поддержка различных селекторов DOM

#### Summary.js
- Отображение резюме с состоянием загрузки
- Обработка ошибок
- Кнопки управления
- Интеграция с настройками

#### Settings.js
- Интерфейс для настройки API ключа
- Валидация ключа
- Инструкции по получению ключа
- Безопасное хранение

3. На странице `chrome://extensions/` найдите ваше расширение

## 🐛 Отладка

### Отладка content script

1. Откройте DevTools на странице Telegram Web
2. Во вкладке Console вы увидите логи от content script
3. Логи включают информацию о парсинге сообщений и API запросах

### Отладка popup

1. Откройте расширение
2. Правый клик на popup > "Проверить элемент"
3. Откроется DevTools для popup

### Отладка background script

1. Перейдите на `chrome://extensions/`
2. Найдите ваше расширение
3. Нажмите "Подробности"
4. В разделе "Фоновый скрипт" нажмите "Проверить представления"

## 🔒 Безопасность

- API ключ хранится локально в chrome.storage
- Ключ не передается на внешние серверы (кроме OpenAI API)
- Все запросы к API выполняются напрямую из content script
- Нет логирования или передачи конфиденциальных данных

## 📝 Известные ограничения

1. **Работает только на web.telegram.org** - Расширение не работает на других сайтах
2. **Требует API ключ OpenAI** - Необходим действующий API ключ для работы
3. **Ограничения API** - Подчиняется лимитам OpenAI API
4. **Парсинг DOM** - Может потребовать обновления при изменении структуры Telegram Web
