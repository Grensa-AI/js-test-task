# Расширение Grensa.AI для Telegram Web

Браузерное расширение, которое анализирует чаты Telegram и генерирует краткие резюме с помощью OpenAI API.

## 🚀 Реализованные функции

### ✅ Основные задачи (обязательные)

- [x] **Парсинг чата и генерация резюме через OpenAI API** - Извлечение сообщений из активного чата и генерация резюме
- [x] **Отображение резюме в окне расширения** - Динамическое отображение сгенерированного резюме
- [x] **Автоматическое обновление при смене чата** - Отслеживание изменений и возможность генерации нового резюме
- [x] **Состояние загрузки с индикатором** - Spinner и информативные сообщения во время обработки

### ✅ Дополнительные задачи

- [x] **История резюме** - Сохранение сгенерированных резюме с возможностью просмотра и удаления
- [x] **Перетаскиваемое окно расширения** - Возможность перемещения окна расширения по экрану с сохранением позиции
- [x] **Обработка аудио сообщений** - Автоматическое получение транскрипций аудио для более полного анализа

## 📦 Установка и настройка

### Предварительные требования

- Node.js версии 16 или выше
- yarn
- Браузер Chrome или любой Chromium-браузер
- Аккаунт OpenAI для получения API ключа

### Установка зависимостей

1. Клонируйте репозиторий:
```bash
git clone https://github.com/YOUR_USERNAME/js-test-task
cd js-test-task
```

2. Установите зависимости:
```bash
yarn install
```

### Настройка API ключа

1. Создайте файл `.env` в корне проекта:
```env
REACT_APP_OPENAI_API_KEY=your_openai_api_key_here
```

2. Получите OpenAI API ключ:
   - Перейдите на сайт [OpenAI](https://platform.openai.com/)
   - Создайте аккаунт или войдите в существующий
   - Перейдите в раздел "API Keys" в личном кабинете
   - Нажмите "Create new secret key"
   - Скопируйте ключ в файл `.env`

⚠️ **Важно**: Никогда не публикуйте API ключ в коде или репозитории!

### Сборка расширения

Для сборки расширения в режиме разработки:
```bash
yarn build
```

Собранные файлы будут находиться в папке `build/`.

### Загрузка расширения в браузер

1. Откройте Chrome/Chromium браузер
2. Перейдите по адресу `chrome://extensions/`
3. Включите "Режим разработчика" в правом верхнем углу
4. Нажмите "Загрузить распакованное расширение"
5. Выберите папку `build/` из вашего проекта
6. Расширение будет установлено и активировано

### Обновление расширения после изменений

1. Внесите изменения в код
2. Выполните сборку: `yarn build`
3. На странице `chrome://extensions/` найдите ваше расширение
4. Нажмите кнопку "Обновить"
5. Перезагрузите страницу Telegram Web для применения изменений

## 🎯 Как использовать

### Основной функционал

1. **Откройте Telegram Web** (https://web.telegram.org)
2. **Выберите чат** для анализа
3. **Прокрутите до интересующих сообщений** - расширение анализирует только видимые на экране сообщения
4. **Нажмите на иконку расширения** в панели инструментов браузера или используйте окно расширения
5. **Нажмите "Создать резюме"** для генерации анализа
6. **Получите резюме** с выделением ключевых фактов, договоренностей и планов

### Дополнительные возможности

#### История резюме
- Все созданные резюме автоматически сохраняются
- Переходите на вкладку "История" для просмотра сохраненных резюме
- Кликните на любое резюме для просмотра в полном размере
- Удаляйте ненужные резюме или очищайте всю историю

#### Перетаскивание окна
- Захватите окно расширения мышью для перемещения по экрану
- Позиция окна автоматически сохраняется между сессиями
- Окно автоматически позиционируется в безопасных границах экрана

#### Обработка аудио
- Расширение автоматически пытается получить транскрипции аудио сообщений
- Информация о количестве обработанных аудио отображается в интерфейсе
- Аудио контент включается в общий анализ чата

## 🔧 Технические особенности

### Использованные технологии
- **React 18** с хуками для управления состоянием
- **Styled Components** для стилизации компонентов
- **Chrome Extensions API** для интеграции с браузером
- **OpenAI API** для генерации резюме
- **Local Storage** для сохранения истории и настроек

### Ключевые архитектурные решения

#### Обход CORS ограничений
Запросы к OpenAI API выполняются через background script, что позволяет обойти CORS ограничения браузера.

#### Парсинг Telegram Web
Расширение анализирует DOM структуру Telegram Web для извлечения сообщений, включая обработку аудио транскрипций.

#### Кеширование запросов
Реализован кеш для предотвращения повторных запросов к API при анализе одних и тех же сообщений.

#### Адаптивное позиционирование
Окно расширения автоматически адаптируется к размеру экрана и сохраняет позицию между сессиями.

## 🐛 Известные ограничения

- Расширение анализирует только сообщения, видимые на экране
- Для анализа аудио требуется, чтобы Telegram Web поддерживал транскрипцию
- API ключ должен быть настроен перед использованием
- Работает только на https://web.telegram.org

## 🔍 Отладка

### Просмотр логов
1. Откройте DevTools на странице Telegram Web (F12)
2. Во вкладке Console вы увидите логи от content script
3. Для отладки popup откройте DevTools на самом расширении (правый клик > "Проверить элемент")

### Типичные проблемы

#### Расширение не появляется
- Убедитесь, что находитесь на https://web.telegram.org
- Проверьте, что расширение активировано в chrome://extensions/
- Перезагрузите страницу Telegram Web

#### Ошибки API
- Проверьте правильность API ключа в .env файле
- Убедитесь, что у вас есть баланс на аккаунте OpenAI
- Проверьте подключение к интернету

#### Не генерируется резюме
- Убедитесь, что в чате есть видимые текстовые сообщения
- Прокрутите чат до места с активным общением
- Проверьте консоль браузера на наличие ошибок
