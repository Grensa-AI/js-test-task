# Решение проблем с доступом к OpenAI API

## 🔧 Проблема: "Sorry, you have been blocked"

Если вы видите сообщение "Sorry, you have been blocked" при попытке зайти на platform.openai.com, это означает, что доступ к сайту заблокирован в вашем регионе или сети.

## 🛠️ Решения:

### 1. **Использование VPN**
- Установите VPN-клиент (NordVPN, ExpressVPN, ProtonVPN и др.)
- Подключитесь к серверу в стране, где OpenAI доступен (США, Канада, Великобритания)
- Попробуйте зайти на platform.openai.com

### 2. **Альтернативные домены**
Попробуйте следующие URL:
- https://api.openai.com/
- https://chat.openai.com/
- https://labs.openai.com/

### 3. **Мобильное приложение**
- Скачайте официальное приложение OpenAI из App Store/Google Play
- Создайте аккаунт через приложение
- Получите API ключ через мобильный интерфейс

### 4. **Корпоративные решения**
- Обратитесь к IT-отделу вашей компании
- Запросите доступ к OpenAI API через корпоративную сеть
- Используйте корпоративные аккаунты

### 5. **Официальные партнеры**
- Microsoft Azure OpenAI Service
- AWS Bedrock
- Google Cloud AI
- Другие официальные партнеры OpenAI

## 🔒 Безопасность

⚠️ **Важно**: 
- Никогда не покупайте API ключи у неофициальных продавцов
- Не используйте поддельные или взломанные ключи
- Храните ключи в безопасном месте
- Не публикуйте ключи в открытом доступе

## 💡 Советы

1. **Попробуйте разные VPN-серверы** - не все серверы работают одинаково
2. **Очистите кэш браузера** - иногда помогает
3. **Используйте режим инкогнито** - для тестирования
4. **Попробуйте мобильный интернет** - иногда мобильные сети не блокируют
5. **Обратитесь к провайдеру** - узнайте о возможности разблокировки

## 🚀 Быстрый старт

Если у вас есть доступ к API ключу:

1. Скопируйте ключ (начинается с `sk-`)
2. Откройте расширение Grensa.AI
3. Нажмите ⚙️ для настроек
4. Вставьте ключ и нажмите "Сохранить"
5. Начните использовать расширение!
